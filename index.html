How to use

  1. Setup the SlipInfo plugin.
  2. Click on the link of the overlay you want to generate
  3. Modify the settings how you want. (Enable/Disable certain sections, change the font, etc)
  4. Click on the "Generate" button
  5. Copy the generated link
  6. Use the generated link in an OBS Browser Source
